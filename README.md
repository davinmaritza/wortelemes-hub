# Wortelemes Portfolio

Personal portfolio website showcasing video content and creative works.

## Features

- Video showcase with YouTube integration
- Portfolio section
- Contact information
- Admin panel for content management

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui components

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
